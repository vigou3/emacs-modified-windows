=== Emacs pour Windows modifié ===

* Si R n'est pas installé à l'endroit habituel (lire: par défaut), il
  peut s'avérer nécessaire de modifier les chemins d'accès dans le
  fichier de configuration système afin que ESS trouve l'exécutable de
  R. Le fichier de configuration système est aisément accessible par
  le raccourci dans le dossier GNU Emacs du menu Démarrer.

* Pour ajouter des extensions à cette distribution de Emacs,
  simplement déposer les fichiers .el dans le dossier
  ...\site-lisp\site-start.d\.

* Voir http://aspell.net/win32/ pour installer des dictionnaires
  additionnels pour Aspell.

* Voir http://gnuwin32.sourceforge.net/ pour installer des applications
  Unix parfois utilisées par Emacs (par exemple: diff, gzip).

* preview-latex requiert une installation de Ghostscript
  (http://www.cs.wisc.edu/~ghost/). S'assurer que le fichier gswin32c.exe
  se trouve dans un des dossiers de la variable d'environnement PATH; voir
  ci-dessous.

* Si elle n'existait pas déjà, une variable d'environnement HOME a été crée
  lors de l'installation. Cette variable est nécessaire pour Emacs et, par
  conséquent, ne doit pas être supprimée. Le dossier HOME est le dossier
  par défaut de Emacs.

* Pour modifier la valeur d'une variable d'environnement (Windows 7):
  dans le menu Démarrer ou dans l'Explorateur Windows, cliquer avec le
  bouton droit sur Ordinateur et choisir Propriétés. Dans la liste de
  gauche du panneau qui s'ouvre alors, cliquer sur Paramêtres système
  avancés, puis sur le bouton Variables d'environnement dans la
  nouvelle fenêtre; sélectionner la variable utilisateur désirée (par
  exemple HOME ou PATH) et appuyer sur Modifier; entrer les
  modifications puis fermer toutes les fenêtres en appuyant sur Ok.

L'information ci-dessus est disponible dans le fichier Updates-fr.txt du
dossier d'installation de Emacs.
